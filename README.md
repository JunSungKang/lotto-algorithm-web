# 로또 제외수 10선 - AI 분석

AI 딥러닝 통계 기반 로또 제외수 분석 서비스

## 🎯 서비스 소개

매 회차마다 **과거 전체 출현 빈도**와 **최근 10회차 추세**를 고급 통계 알고리즘으로 분석하여, 
다음 회차에서 출현 확률이 가장 낮은 **제외수 10개**를 제공합니다.

## 🔒 보안

- 분석 알고리즘은 Private Repository에서 관리
- 웹사이트에는 분석 결과만 표시 (로직 비공개)
- GitHub Actions를 통한 자동 업데이트

## 🚀 기술 스택

- React + Vite
- GitHub Pages
- AI 통계 분석 엔진 (Server-Side)
